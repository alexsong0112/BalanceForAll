
# About the project:
BalanceForAll is a website that we created that focuses on SDG 5 to raise awareness on gender equality. It focuses on many real world aspects like stereotypes, discrimination, and unequal opportunities. Our main goal for this website was to help people understand these problems that people experience and to encourage a positive change.

# Purpose
The purpose of this website was to educate users about gender inequality using real examples, facts, and interactive pieces like the quiz.

# Pages
- Home Page - Introduction to gender equality
- Visual Stories - Real-life examples shown through images and short explanations
- Gender in Sports - Differences between men’s and women’s sports
- Gender in Workforce - Workplace inequality and pay gap
- Linked Goals - Connection to UN Sustainable Development Goals
- Are You Equal? - Interactive quiz
- Join the Change - Ways to support equality

# How was it made
We built this website using Pycharm for coding and using Github for collaboration. The website was built on HTML code and CSS for the style sheet.
